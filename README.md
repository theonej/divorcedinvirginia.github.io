# Divorced in Virginia

This is a GitHub Pages website built with Jekyll.

## Getting Started

- Edit `_config.yml` to customize site settings
- Update `index.html` or add markdown files
- Push changes to GitHub to deploy

## Local Development

To run locally:
```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`
